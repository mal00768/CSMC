Detailed Description

This class describes the pairing hamiltonian defined as

The class is templated on the unsigned integer data type chosen to represent a state in the configuration space. This can be uint32_t, uint64_t, or uint128_t depending on the architecture and compiler support of the host system.

The constructor requires as input the number of levels (levels), the number of pairs (NP), an std::vector containing the set of single particle energies (Esp), and lastly a two dimensional std::vector containing the set of pairing matrix elements (Gij).
