Function definitions of random number class. Uses mersenne twister algorithm.

Random number class. Uses mersenne twister algorithm.

This is the Walker class which handles the actual random evolution of the ensemble.

The class is templated on the unsigned integer data type chosen to represent a state in the configuration space. This can be uint32_t, uint64_t, or uint128_t depending on the architecture and compiler support of the host system.
