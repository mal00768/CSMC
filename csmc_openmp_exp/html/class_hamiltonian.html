Detailed Description

template<typename walker>
class Hamiltonian< walker >

Hamiltonian class. walker type must be an unsigned type like uint32_t or uint64_t!

Parameters
NPinteger value for the number of pairs.
levelsinteger value for the number of levels.
Espstd::vector containing the single particle energies.
Gij2-dimensional std::vector containing all the two body matrix elements Gij.
nbitsinteger value for the number of bits in the walker data type.
Eshiftdouble value for the shift in the diagonal matrix elements.s

The constructor requires as input the number of levels (levels), the number of pairs (NP), an std::vector containing the set of single particle energies (Esp), and lastly a two dimensional std::vector containing the set of pairing matrix elements (Gij).

double Hamiltonian< walker >::Probdiagonal ( const std::vector< walker > &  rstate) const

This function returns the probability of making a diagonal transition in configuration space i.e. where a state transitions into the same state and no pairs are moved.

In general the set of transition probabilites are proportional to the normalized matrix elements of all states connected to the current configuration.

where following the linear normalization condition.

This function returns the probability of making a off-diagonal transition in configuration space i.e. where a state transitions into a new state and a pair is moved from an occupied level to an unoccupied level.
